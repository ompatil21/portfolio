## ü§ñ Introduction

This repository contains the source code for my personal portfolio website.

I use this site as a central hub to:

- present my **software engineering & full-stack projects**  
- highlight my experience with **Next.js, Python/Flask, AWS, and analytics**  
- give recruiters and collaborators a quick, clean overview of who I am and what I build.

The design is focused on being:

- **fast** (Next.js, image optimisation)  
- **interactive** (Framer Motion, smooth animations)  
- **developer-focused** (showcasing real projects, not just placeholders)

## üîã Features

Some of the key sections & interactions in this portfolio:

- **Hero Section**  
  Short, clear introduction with a subtle spotlight / gradient background and animated entry effects.

- **Bento-style About Grid**  
  A grid-based layout that highlights:
  - who I am  
  - how I work  
  - my tech stack  
  - call-to-action to start a project together

- **Projects Section**  
  Cards showing my main work with:
  - concise description  
  - tech icons (stack at a glance)  
  - links to GitHub repositories or live demos

- **Experience & Highlights**  
  A section to summarise roles, responsibilities, and areas I focus on (full-stack, cloud, analytics, healthcare tools, etc.).

- **Testimonials / Social Proof (optional)**  
  Space for testimonials, recommendations, or social proof as I collect them.

- **Smooth Animations & Responsiveness**  
  Fully responsive layout with:
  - animated transitions between sections  
  - hover effects on cards  
  - proper mobile/tablet/desktop behaviour
